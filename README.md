# Lachie Sims — Portfolio

My personal portfolio site, built with Next.js, Tailwind CSS, and Shadcn UI.

## Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS 4.0
- **Components:** Shadcn UI
- **Language:** TypeScript

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
